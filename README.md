# SSC CHSL Study Planner

Personalized weekly study schedule planner for SSC CHSL exam preparation.

## 🚀 Quick Start

```bash
npm install
npm start
```

App runs at: http://localhost:3000

## 📦 Deploy to Vercel

1. Push code to GitHub
2. Import project in Vercel
3. Deploy (no environment variables needed)

## ✨ Features

- 📅 **Personalized Weekly Schedule** - Monday to Sunday routine
- 📊 **Analytics Dashboard** - Study hours, time allocation, location split
- 📚 **Subject Coverage** - Reasoning, Maths, English, GK, Science, Social Studies
- 💪 **Fitness Integration** - Daily workout sessions
- 💖 **Work-Life Balance** - Includes personal time, meals, sleep
- 🎨 **Modern UI** - Dark theme with gradient effects

## 🏗️ Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- 100% Frontend (No Backend Required)
